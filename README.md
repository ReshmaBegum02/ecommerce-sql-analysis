# E-commerce SQL Data Analysis Project

## Project Overview

This project demonstrates SQL skills by analyzing an E-commerce database.
The project includes database schema creation, data insertion, and business analysis queries.

## Database Structure

The database contains the following tables:

* Customers
* Products
* Orders
* Order Items

These tables are connected using **Primary Keys** and **Foreign Keys**.

## Files in This Project

schema
Contains SQL code to create database tables.

data
Contains SQL code to insert sample data into tables.

queries
Contains basic SQL queries for analysis.

advanced_queries
Contains advanced SQL queries such as joins, aggregations, and business insights.
